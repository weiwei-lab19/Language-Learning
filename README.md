# Language-Learning
Repository for learning new languages

Link to webpage: https://weiwei-lab19.github.io/Language-Learning/

## Adding vocabulary
Add or edit entries in [vocabulary.js](vocabulary.js). Each item should use this shape:

```js
{ word: '안녕하세요', translation: 'hello', pronunciation: 'annyeonghaseyo', lang: 'Korean' }
```

Update the file, commit the change, and the site will pick up the new vocabulary when it is published.
