# Language-Learning
Repository for learning new languages

## Simple GitHub Pages site
This repository includes a simple static webpage to help learn vocabulary: `index.html`, `styles.css`, and `script.js`.

To publish with GitHub Pages (site served from the repository root):

1. Push the repository to GitHub.
2. In the repository Settings → Pages, set the source to the `main` branch (or whichever branch you use) and the root folder.
3. Save and visit the provided URL (usually `https://<username>.github.io/<repo>`).

The page provides a searchable index and alphabetical buckets for quick browsing. Vocabulary is stored in the browser's `localStorage` so you can add and keep words locally.
